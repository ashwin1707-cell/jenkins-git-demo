\# Jenkins CI/CD Pipeline Project 🚀



\## 📌 Overview

This project demonstrates a simple CI pipeline using:



\- Git

\- GitHub

\- Jenkins (Pipeline as Code)

\- Docker (Jenkins running in container)



\## ⚙️ What This Pipeline Does



1\. Pulls source code from GitHub

2\. Runs Build stage

3\. Runs Test stage

4\. Executes inside Linux-based Jenkins container
